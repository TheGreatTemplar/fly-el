Fly El site prototype v2

Статический сайт на HTML/CSS.
Структура сделана по мотивам простой лабораторной страницы, но оформление адаптировано под эмблемы Fly El: светлый фон, сине-графитовая палитра, технические блоки и отдельный раздел для тестирования радиокомпонентов.

Файлы:
- index.html — главная страница
- style.css — оформление
- pages/components.html — тестирование радиокомпонентов
- pages/about.html — о проекте
- pages/projects.html — проекты
- pages/articles.html — статьи
- pages/archive.html — архив
- pages/contacts.html — контакты
- assets/img/ — логотипы

Для GitHub Pages достаточно загрузить содержимое этой папки в репозиторий.
